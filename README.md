# 💪 Apex Gym Website

A premium, high-performance gym website designed for modern fitness brands. Built with bold visuals, immersive animations, and a responsive user experience, this template helps gyms create a powerful online presence that motivates visitors from the very first click.

---

## ✨ Overview

Apex Gym is crafted to showcase strength, energy, and professionalism through modern web design. With dynamic layouts, engaging interactions, and optimized performance, the website is designed to convert visitors into members while reflecting the identity of a premium fitness brand.

---

## 🚀 Features

- Premium modern UI/UX
- Fully responsive design
- Smooth animations and interactive elements
- Bold hero section
- Membership plans
- Trainer showcase
- Class schedules
- Success stories & testimonials
- Gallery
- Contact & location section
- Fast loading and optimized performance

---

## 🛠️ Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui
